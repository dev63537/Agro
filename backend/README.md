# Agro Billing SaaS - Backend

This repository contains the backend for the Multi-Tenant Agro Billing SaaS.

## Tech stack

- Node.js + Express
- MongoDB (Mongoose)
- JWT Authentication
- AWS S3 (optional) or local storage
- pdfkit for PDF invoice generation
- Jest + Supertest for tests

## Getting started (development)

1. Copy `.env.example` to `.env` and fill required values.
2. Install dependencies:

```bash
npm install
